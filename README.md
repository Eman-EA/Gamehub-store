# 🎮 GameHub Store

A front-end gaming e-commerce store built with **HTML, CSS, and JavaScript**.  
This project was created as a practice to showcase my front-end development skills.

## 🚀 Live Demo
👉 [View Live Website](https://eman-ea.github.io/Gamehub-store/)

## ✨ Features
- ⭐ Featured Products section with gaming gear (headsets, mice, keyboards, etc.)
- 🔥 Flash Sale banner with a countdown timer
- 🔍 Search bar for finding products
- 🛒 "Add to Cart" buttons (UI only)
- 📱 Fully responsive design (works on mobile and desktop)

## 🛠️ Technologies Used
- **HTML5** - Structure
- **CSS3** - Styling & Animations
- **JavaScript (Feather Icons)** - Icons

## 📝 What I Learned
- How to structure a multi-section e-commerce layout.
- How to use Flexbox and Grid for responsive design.
- How to deploy a static website using **GitHub Pages**.
